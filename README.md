## Dependencies

The matrix abstract layer depends on several packages which
have to be available on your machine.

 - ROS stacks:
   - redundant_manipulator_control
 - System tools:
   - CMake (>= 2.6)
   - pkg-config
   - usual compilation tools (GCC/G++, make, etc.)
